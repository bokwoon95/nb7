What is a post slug?

A post slug is the ending part of a URL that describes the post in a few words. Some examples:

this is the post slug => https://example.com/posts/this-is-the-post-slug/
my day trip to hokkaido => https://example.com/posts/my-day-trip-to-hokkaido/

If a post slug is not explicitly provided, the title of the post (the first line) will be used as the slug.
